# 이 레포지토리는

CS 면접을 위해 공부하는 레포지토리입니다.

## 뭘 남기나요

1. 오늘의 질문
2. 질문에 대한 내 생각
3. 답을 본 뒤 알게 된 것
4. 틀렸거나 부족했던 부분
5. 면접용 1분 답변
6. 한 줄 요약

아직 초기 단계이므로 빠르게 시작하고 고민해서 띄워 올리는 것을 목표로 합니다.

## 파일 규칙

새 공부 노트는 과목 폴더 안에 만듭니다.

```text
YYYY-MM-DD-주제.md
```

예시:

```text
01_network/2026-07-07-주소창에-url을-입력하면-일어나는-과정.md
```

파일명은 자동화가 처리하기 쉽도록 날짜와 주제를 포함합니다. 요일은 본문이나 README에 적고, 파일명에는 넣지 않는 것을 기본값으로 둡니다.

## 문서 역할

- 과목별 노트: 공부한 내용의 원본입니다.
- 과목별 `README.md`: 해당 과목에서 공부한 주제 목록입니다.
- `00-daily-report`: 그날 공부한 원본 노트를 빠르게 보기 위한 복사본입니다.
- `00_docs/templates`: AI가 새 노트를 만들 때 참고하는 템플릿입니다.
- `00_docs/workflow.md`: 공부 시작과 마무리 자동화 규칙입니다.
