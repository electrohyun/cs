## 공부 방식

1. 오늘의 주제와 함께 템플릿 생성을 요청합니다.
2. AI가 과목 폴더에 `YYYY-MM-DD-주제.md` 형식으로 노트를 만듭니다.
3. 해당 노트에서 공부를 마칩니다.
4. 마무리를 요청하면 AI가 과목 README와 데일리 리포트를 갱신합니다.

자세한 진행 규칙은 [workflow](./00_docs/workflow.md)를 따릅니다.
